## ğŸ“ Exam Alignment

Each project maps to official Microsoft exam objectives:

### AZ-104: Azure Administrator Associate
- Manage identities and governance (15-20%)
- Implement and manage storage (15-20%)
- Deploy and manage compute (20-25%)
- Configure and manage networking (20-25%)
- Monitor and maintain resources (10-15%)

### AZ-305: Azure Solutions Architect Expert
- Design identity, governance, and monitoring (25-30%)
- Design data storage solutions (20-25%)
- Design business continuity solutions (15-20%)
- Design infrastructure solutions (30-35%)

### AI-102: Azure AI Engineer Associate
- Plan and manage Azure AI solutions (15-20%)
- Implement decision support solutions (10-15%)
- Implement computer vision solutions (15-20%)
- Implement NLP solutions (25-30%)
- Implement knowledge mining solutions (15-20%)
- Implement conversational AI (15-20%)
